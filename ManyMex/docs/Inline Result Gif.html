Inline Result Gif

Represents a link to an animated GIF file. By default, this animated GIF file will be sent by the user with optional caption. Alternatively, you can use input_message_content to send a message with the specified content instead of the animation.

Field Type Optional Description
type String No Type of the result, must be gif
id String No Unique identifier for this result, 1-64 bytes
gif_url or gif_file_id String No A valid URL for the GIF file. File size must not exceed 1MB or a valid file identifier for the GIF file
gif_width Integer Yes Width of the GIF
gif_height Integer Yes Height of the GIF
gif_duration Integer Yes Duration of the GIF
thumb_url String No URL of the static thumbnail for the result (jpeg or gif)
title String Yes Title for the result
caption String Yes Caption of the GIF file to be sent, 0-1024 characters
parse_mode String Yes Send Markdown or HTML, if you want Telegram apps to show bold, italic, fixed-width text or inline URLs in the media caption.
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the GIF animation
