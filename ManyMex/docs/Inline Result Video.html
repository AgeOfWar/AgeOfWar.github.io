Inline Result Video

Represents a link to a page containing an embedded video player or a video file. By default, this video file will be sent by the user with an optional caption. Alternatively, you can use input_message_content to send a message with the specified content instead of the video.

Field Type Optional Description
type String No Type of the result, must be video
id String No Unique identifier for this result, 1-64 bytes
video_url video_file_id String No A valid URL for the embedded video player or video file or a valid file identifier for the video file
mime_type String No Mime type of the content of video url, "text/html" or "video/mp4"
thumb_url String No URL of the thumbnail (jpeg only) for the video
title String No Title for the result
caption String Yes Caption of the video to be sent, 0-1024 characters
parse_mode String Yes Send Markdown or HTML, if you want Telegram apps to show bold, italic, fixed-width text or inline URLs in the media caption.
video_width Integer Yes Video width
video_height Integer Yes Video height
video_duration Integer Yes Video duration in seconds
description String Yes Short description of the result
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the video. This field is required if InlineQueryResultVideo is used to send an HTML-page as a result (e.g., a YouTube video).
