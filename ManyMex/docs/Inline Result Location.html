Inline Result Location

Represents a location on a map. By default, the location will be sent by the user. Alternatively, you can use input_message_content to send a message with the specified content instead of the location.

Field Type Optional Description
type String No Type of the result, must be location
id String No Unique identifier for this result, 1-64 Bytes
latitude Float number No Location latitude in degrees
longitude Float number No Location longitude in degrees
title String No Location title
live_period Integer Yes Period in seconds for which the location can be updated, should be between 60 and 86400.
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the location
thumb_url String Yes Url of the thumbnail for the result
thumb_width Integer Yes Thumbnail width
thumb_height Integer Yes Thumbnail height
