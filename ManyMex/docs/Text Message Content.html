Text Message Content

Represents the content of a text message to be sent as the result of an inline query.

Field Type Optional Description
message_text String No Text of the message to be sent, 1-4096 characters
parse_mode String Yes Send Markdown or HTML, if you want Telegram apps to show bold, italic, fixed-width text or inline URLs in your bot's message
disable_web_page_preview Boolean Yes Disables link previews for links in the sent message
