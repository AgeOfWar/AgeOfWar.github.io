Configuration

This is how your JSON config should be configured:

Field Type Optional Description
on_message Array of Message Handler yes List of message handlers
welcome_message Array of String yes List of messages to send when an user joins a chat, randomly chosen, 1-4096 characters. The following tags are currently supported: <b>bold</b>, <strong>bold</strong>, <i>italic</i>, <em>italic</em>, <a href="http://www.example.com/">inline URL</a>, <a href="tg://user?id=123456789">inline mention of a user</a>, <code>inline fixed-width code</code>, <pre>pre-formatted fixed-width code block</pre>. Use {0} to mention the new chat member.
on_callback Array of Callback Handler yes List of callback handlers
on_inline Array of Inline Handler yes List of inline handlers
