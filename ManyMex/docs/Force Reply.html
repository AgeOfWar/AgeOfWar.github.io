Force Reply

Upon receiving a message with this object, Telegram clients will display a reply interface to the user (act as if the user has selected the bot‘s message and tapped ’Reply').

Field Type Optional Description
force_reply True no Shows a reply interface to the user, as if they manually selected the bot‘s message and tapped ’Reply'
selective Boolean yes Use this parameter if you want to force reply from specific users only. Targets: 1) users that are @mentioned in the text of the Message object; 2) if the bot's message is a reply, sender of the original message
