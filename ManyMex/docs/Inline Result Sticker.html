Inline Result Sticker

Represents a link to a sticker stored on the Telegram servers. By default, this sticker will be sent by the user. Alternatively, you can use input_message_content to send a message with the specified content instead of the sticker.

Field Type Optional Description
type String No Type of the result, must be sticker
id String No Unique identifier for this result, 1-64 bytes
sticker_file_id String No A valid file identifier of the sticker
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the sticker
