Inline Result Audio

Represents a link to an mp3 audio file stored on the Telegram servers. By default, this audio file will be sent by the user. Alternatively, you can use input_message_content to send a message with the specified content instead of the audio.

Field Type Optional Description
type String No Type of the result, must be audio
id String No Unique identifier for this result, 1-64 bytes
audio_url or audio_file_id String No A valid URL for the audio file or a valid file identifier for the audio file
caption String Yes Caption, 0-1024 characters
performer String Yes Performer
audio_duration Number Yes Audio duration in seconds
parse_mode String Yes Send Markdown or HTML, if you want Telegram apps to show bold, italic, fixed-width text or inline URLs in the media caption.
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the audio
