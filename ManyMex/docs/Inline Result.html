Inline Result

This object represents one result of an inline query. Telegram clients currently support results of the following types:

  • Inline Result Audio
  • Inline Result Document
  • Inline Result Gif
  • Inline Result Mpeg4Gif
  • Inline Result Photo
  • Inline Result Sticker
  • Inline Result Video
  • Inline Result Voice
  • Inline Result Article
  • Inline Result Contact
  • Inline Result Game
  • Inline Result Location
  • Inline Result Venue
