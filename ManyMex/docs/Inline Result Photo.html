Inline Result Photo

Represents a link to a photo. By default, this photo will be sent by the user with optional caption. Alternatively, you can use input_message_content to send a message with the specified content instead of the photo.

Field Type Optional Description
type String No Type of the result, must be photo
id String No Unique identifier for this result, 1-64 bytes
photo_url or photo_file_id String No A valid URL of the photo. Photo must be in jpeg format. Photo size must not exceed 5MB or a valid file identifier of the photo
thumb_url String No URL of the thumbnail for the photo
photo_width Integer Yes Width of the photo
photo_height Integer Yes Height of the photo
title String Yes Title for the result
description String Yes Short description of the result
caption String Yes Caption of the photo to be sent, 0-1024 characters
parse_mode String Yes Send Markdown or HTML, if you want Telegram apps to show bold, italic, fixed-width text or inline URLs in the media caption.
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the photo
