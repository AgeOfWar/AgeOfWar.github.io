Inline Result Voice

Represents a link to a voice recording in an .ogg container encoded with OPUS. By default, this voice recording will be sent by the user. Alternatively, you can use input_message_content to send a message with the specified content instead of the the voice message.

Field Type Optional Description
type String No Type of the result, must be voice
id String No Unique identifier for this result, 1-64 bytes
voice_url or voice_file_id String No A valid URL for the voice recording or a valid file identifier for the voice message
title String No Recording title
caption String Yes Caption, 0-1024 characters
parse_mode String Yes Send Markdown or HTML, if you want Telegram apps to show bold, italic, fixed-width text or inline URLs in the media caption.
voice_duration Integer Yes Recording duration in seconds
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the voice recording
