Inline Result Mpeg4Gif

Represents a link to a video animation (H.264/MPEG-4 AVC video without sound). By default, this animated MPEG-4 file will be sent by the user with optional caption. Alternatively, you can use input_message_content to send a message with the specified content instead of the animation.

Field Type Optional Description
type String No Type of the result, must be mpeg4_gif
id String No Unique identifier for this result, 1-64 bytes
mpeg4_url or mpeg4_file_id String No A valid URL for the MP4 file. File size must not exceed 1MB or a valid file identifier for the MP4 file
mpeg4_width Integer Yes Video width
mpeg4_height Integer Yes Video height
mpeg4_duration Integer Yes Video duration
thumb_url String No URL of the static thumbnail (jpeg or gif) for the result
title String Yes Title for the result
caption String Yes Caption of the MPEG-4 file to be sent, 0-1024 characters
parse_mode String Yes Send Markdown or HTML, if you want Telegram apps to show bold, italic, fixed-width text or inline URLs in the media caption.
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the video animation
