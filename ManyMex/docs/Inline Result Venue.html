Inline Result Venue

Represents a venue. By default, the venue will be sent by the user. Alternatively, you can use input_message_content to send a message with the specified content instead of the venue.

Field Type Optional Description
type String No Type of the result, must be venue
id String No Unique identifier for this result, 1-64 Bytes
latitude Float No Latitude of the venue location in degrees
longitude Float No Longitude of the venue location in degrees
title String No Title of the venue
address String No Address of the venue
foursquare_id String Yes Foursquare identifier of the venue if known
foursquare_type String Yes Foursquare type of the venue, if known. (For example, "arts_entertainment/default", "arts_entertainment/aquarium" or "food/icecream".)
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the venue
thumb_url String Yes Url of the thumbnail for the result
thumb_width Integer Yes Thumbnail width
thumb_height Integer Yes Thumbnail height
