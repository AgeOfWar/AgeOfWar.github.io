Inline Result Article

Represents a link to an article or web page.

Field Type Optional Description
type String No Type of the result, must bearticle
id String No Unique identifier for this result, 1-64 Bytes
title String No Title of the result
input_message_content Message Content No Content of the message to be sent
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
url String Yes URL of the result
hide_url Boolean Yes Pass True, if you don't want the URL to be shown in the message
description String Yes Short description of the result
thumb_url String Yes Url of the thumbnail for the result
thumb_width Integer Yes Thumbnail width
thumb_height Integer Yes Thumbnail height
