Inline Result Contact

Represents a contact with a phone number. By default, this contact will be sent by the user. Alternatively, you can use input_message_content to send a message with the specified content instead of the contact.

Field Type Optional Description
type String No Type of the result, must be contact
id String No Unique identifier for this result, 1-64 Bytes
phone_number String No Contact's phone number
first_name String No Contact's first name
last_name String Yes Contact's last name
vcard String Yes Additional data about the contact in the form of a vCard, 0-2048 bytes
reply_markup Inline Keyboard Yes Inline keyboard attached to the message
input_message_content Message Content Yes Content of the message to be sent instead of the contact
thumb_url String Yes Url of the thumbnail for the result
thumb_width Integer Yes Thumbnail width
thumb_height Integer Yes Thumbnail height
