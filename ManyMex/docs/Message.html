Message

Message to send.

Field Type Optional Description
text Array of String yes List of texts of the message to send, randomly chosen, 1-4096 characters. The following tags are currently supported: <b>bold</b>, <strong>bold</strong>, <i>italic</i>, <em>italic</em>, <a href="http://www.example.com/">inline URL</a>, <a href="tg://user?id=123456789">inline mention of a user</a>, <code>inline fixed-width code</code>, <pre>pre-formatted fixed-width code block</pre>
files Array of String yes List of paths to file to send, randomly chosen
send_as_reply Boolean yes If true this message will be a reply to the handled message
reply_markup Inline Keyboard or
Reply Keyboard or
Remove Reply Keyboard or
